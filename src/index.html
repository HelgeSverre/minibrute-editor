MiniBrute Editor

MIDI Channel Select

Sets the MIDI channel for outgoing messages. Choose between channels 1-16.

Sets the MIDI channel the instrument will respond to. Options: 1-16, All (responds to any channel).

Keyboard Parameters

Sets the audio level required to trigger the envelopes. Options: High, Medium, Low.

Determines if the LFO resets when a key is pressed. Options: On (resets), Off (runs freely).

Module Parameters

Sets note priority when more than one key is pressed. Options: High, Low, Last.

Sets the velocity curve. Options: Lin (linear), Log (logarithmic), Anti Log (exponential).

Enables legato mode. On: envelopes are not retriggered. Off: envelopes are retriggered on each key press.

Sets the aftertouch curve. Options: Exponential, Logarithmic, Linear.
